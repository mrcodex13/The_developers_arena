# Sales Data Analysis Project

This project performs a complete data analysis workflow using the provided sales dataset in the data folder.

## What this project includes

- Data loading and validation
- Data cleaning and preparation
- Descriptive analysis of sales performance
- Two visualizations created with matplotlib
- A written report with business insights

## Project structure

- main.py: main analysis script
- data/sales_data.csv: source dataset
- visualizations/: generated charts
- report/: generated analysis report

## Requirements

Install the dependencies with:

```bash
pip install -r requirements.txt
```

## Run the analysis

From this folder, run:

```bash
python main.py
```

The script will generate:

- visualizations/product_sales_bar.png
- visualizations/monthly_sales_trend.png
- report/sales_analysis_report.md
