# Student Grade Calculator

## 📌 Project Description

This is a simple Python program that calculates a student's grade based on the marks entered by the user. The program validates the input, assigns grades using conditional statements, and displays encouraging messages based on the student's performance.

---

## 🚀 Features

- Accepts student name and marks as input.
- Validates marks to ensure they are between 0 and 100.
- Uses a `while` loop to handle invalid inputs.
- Uses `if-elif-else` statements for grading logic.
- Uses a function to calculate grades and messages.
- Displays encouraging messages for each grade.

---

## 📝 Grading System

| Marks    | Grade |
| -------- | ----- |
| 90 - 100 | A     |
| 80 - 89  | B     |
| 70 - 79  | C     |
| 60 - 69  | D     |
| 0 - 59   | F     |

---

## 💻 How to Run

1. Make sure Python 3 is installed on your system.
2. Open a terminal in the project directory.
3. Run the following command:

```bash
python grade_calculator.py
```
